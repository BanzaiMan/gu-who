What is GU-Who?

GU-Who is a script to audit public members of GitHub Organisations. It checks the state of all the users, and raises a GitHub issue for anyone who does not meet one or more of the following requirements:

  • Enabled two factor authentication
  • Sponsorship (an entry in users.txt file)
  • Has filled out the fullname field on their account

Pre-requisites

Your org must have a repo called 'People'. For security purposes, we recommend this to be private as all the (potentially vulnerable) members of your org will be visible on this repo.
